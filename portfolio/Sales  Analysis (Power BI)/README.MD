# Strategic Business Insights Dashboard: Monthly Sales Operations

## Project Overview
This project features a robust Power BI dashboard designed to analyze and visualize monthly sales data. By centralizing key metrics, the dashboard provides a clear narrative on revenue growth, regional dominance, and product lifecycle performance.

## Technical Stack
* **Tool:** Power BI Desktop
* **Data Source:** Monthly Sales [Dataset](https://drive.google.com/drive/folders/13RsNqZ0OWr07VqofAHEoLqbLtcgMtHOf) (Excel/CSV)
* **Key Visuals:** Ribbon Charts, Maps, Slicers, Donut Charts, and KPIs.

## Key Features & Insights
* **Trend Analysis:** Used Ribbon Charts to visualize the rank change of categories over time, identifying which products are gaining market share.
* **Geospatial Insights:** Implemented Map visuals to pinpoint underperforming territories for strategic intervention.
* **Categorical Deep-Dive:** Analyzed Sales by Sub-Category to determine the "Pareto" drivers of total revenue.
* **Interactive Filtering:** Integrated multi-select slicers for seamless data exploration across different time periods and regions.
